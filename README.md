# 🚀 C++ Utility Library (clsUtil)

## 📖 Project Overview
A professional, header-only C++ Utility Library designed for modularity and high performance. It provides a robust collection of static helper functions for random generation, array manipulation, memory swapping, and text encryption. The library is built as a **static class (clsUtil)**, enabling seamless integration into any project without object instantiation overhead.

## ⚙️ Library Capabilities

### 🎲 Randomization Engine
*   Generate random numbers, characters, and words.

### 📦 Array Management
